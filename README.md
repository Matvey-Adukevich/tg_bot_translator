### Телеграм бот для изучения иностранных языков(английский)
* Инструмент для ежедневного изучения английского
### Функции:
* "Слово дня" - каждый день вы будете узнавать новое слово
* Перевод - подробный перевод слова, его синонимы и антонимы, словосочетания с этим словом, пример употребления этого слова в предложении, правильное произношение
* Игра "Синонимы-Антонимы" - выберите синоним к слову из 4 вариантов
### Run: ```python main.py```
